The Project

The goal is to create a repository of good resources for those who are starting a new programming language. As someone who is interested in programming, finding the best learning materials for a language is important when I have limited time and want to learn quickly.

Language Overview

Each language posted to this website will have two sections. One section will be around a Hello World program for the language. The second section will offer good resources for continued learning of the language.
